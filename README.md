# AerialViews — Custom Font Fork

A personal fork of [AerialViews](https://github.com/theothernt/AerialViews) with additional font support

## What's changed

- Added custom fonts as selectable options
- Available in **Settings → Appearance → Typography**

### Fonts included
- **MiSans** — native Xiaomi font, great on Mi TV
- More fonts coming soon!

## How to build

This repo uses GitHub Actions to build automatically.

1. Fork or clone this repo
2. Push any commit to `master`
3. Go to the **Actions** tab on GitHub
4. Wait for the build to finish (~10 minutes)
5. Download the APK from the **Artifacts** section of the run
6. Use the `fdroid` flavor — it works best for sideloading
